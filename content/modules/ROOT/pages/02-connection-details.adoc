== Lab structure (18 minutes total)

This lab follows a *see it first, then build it* approach:

. *Get Started* (3 min) -- The app is already deployed. Explore it, hit the URL, see that it works.
. *Explore* (5 min) -- Inspect the three resources OpenShift created: Pod, Service, and Route.
. *Build It Yourself* (10 min) -- Delete the demo app and rebuild it from scratch using the CLI or the Web interface

== Your lab environment

NOTE: In OpenShift, a "project" is like a "namespace" in Kubernetes, and it acts as a boundery to allow users to deploy 
workloads and admins to restrict permissions, quota and other elements.
