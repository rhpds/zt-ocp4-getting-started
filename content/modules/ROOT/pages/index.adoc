= OpenShift Getting Started

Welcome to the *OpenShift Getting Started* lab series.

These labs are designed for developers, platform engineers, and DevOps engineers who are new to OpenShift.
Each lab is hands-on and self-contained: you'll work directly on a live OpenShift cluster.

== Labs in this series

* xref:lab-01-deploy-the-app.adoc[Lab 1: Deploy the App] +
  Deploy an application 3 different ways: from a container image, via Source-to-Image, and with a Helm chart. Learn the core building blocks: Pods, Services, and Routes.

* xref:lab-02-apps-never-down.adoc[Lab 2: Apps That Never Go Down] +
  Configure health probes, scale to multiple replicas, and set up autoscaling to handle real traffic.

* xref:lab-04-gitops.adoc[Lab 3: Git Is Your Source of Truth] +
  Register your application under GitOps management and watch ArgoCD keep the cluster in sync with Git.
