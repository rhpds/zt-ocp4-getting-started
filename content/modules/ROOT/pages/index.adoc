= OpenShift Getting Started

Welcome to the *OpenShift Getting Started* lab.

This lab is designed for developers, platform engineers, and DevOps engineers who are new to OpenShift.
The lab is hands-on and self-contained: you'll work directly on a live OpenShift cluster.

ifeval::["{lab_number}" == "1"]
== What You'll Learn

Deploy an application 3 different ways: from a container image, via Source-to-Image, and with a Helm chart. Learn the core building blocks: Pods, Services, and Routes.
endif::[]

ifeval::["{lab_number}" == "2"]
== What You'll Learn

Configure health probes, scale to multiple replicas, and set up autoscaling to handle real traffic. Understand how OpenShift keeps your applications running and resilient.
endif::[]

ifeval::["{lab_number}" == "3"]
== What You'll Learn

Register your application under GitOps management and watch ArgoCD keep the cluster in sync with Git. Learn how to use Git as the source of truth for your application deployments.
endif::[]
